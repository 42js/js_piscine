초기값과 임의의 갯수의 함수를 매개변수로 받고 순서대로 실행하는 `ft.go()` 라는 함수를 만들어보세요. 해당 함수의 프로토타입은 이렇게 생겼습니다. 

```javascript
ft.go(...args); // 사용시 ft.go(init, func1, func2, ... )
```

초기값 init 이 func1 의 매개변수가 되고 그 리턴값이 func2 의 매개변수가 되는 식으로 작동합니다. 
```javascript
ft.go(0, 
	(a) => a + 1, 
	(a) => a + 10, 
	(a) => a + 100, 
	(a) => a + 1000, 
	ft.log);
```
```
// 결과
1111
```
