여러분들은 `쿠키런 킹덤` 의 출입국 심사를 담당하는 직원이 되셨습니다!

`쿠키런 킹덤` 으로 들어오기 위해서는 특별한 쿠키가 필요한데 해당 쿠키가 맞는지 확인을 하고 올바르다면 환영하는 응답을 보내주어야 합니다!

아래의 제약 조건을 지키며 구현해 주세요!

### API 명세

- **URL**
  `/passport`
- **Method**
  `GET`
- **Params**
  - None
- **Query**
  - None
- **Body**
  - None
- **Cookies**
  - name: `special_cookie`
  - value: `42`
- Cookie 조건이 맞을 시
  - Code: `200`
  - content
    ```json
    🍪 쿠키런 킹덤에 오신 걸 환영 합니다~! 🍪
    ```
- 잘못된 요청 시
  - Code: `401`
  - content
    ```json
    🍪 쿠키런 킹덤에는 아무나 출입 할 수 없습니다!! 🍪
    ```
